# Intro

This app will alow users to interact with a local ollama chatbot users will have the abelty to install ollama easely via the gui or link it to a url and then use the chatbot to interact with it also there will be 3d carachter showing on screen acting as the ai

---

## Must Have

- local ollama interaction
- ollama memory
- 3d vroid carachter showing
- gpt sovits integration

## Should Have

- idle animation for caracter
- lip syncing
- simple docker install

## Could Have

- Ollama visuel install
- Customizable Character Options
- Background Environments
- Mic input

## Won’t Have

- ai cloud integration like chat gpt
- more tts integration with previders like elven labs

# Technical Specifications

This project will use **Python Flask** and will be installable via **Docker**. It will also use a **database** to store tasks and user data. also it will use react
